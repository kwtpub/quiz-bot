Простой CLI для тренировки билетов и отслеживания прогресса.

## Требования
- Node.js 18+

## Установка
```bash
npm install
```

## Сборка
```bash
npm run build
```

## Запуск
```bash
npm start
```

## Данные
- Билеты: `src/data/tickets.json`
- История (создаётся автоматически): `src/data/history.json`
